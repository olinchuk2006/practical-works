# practical-works
> Найгарніший цифровий каталог рідкісних україномовних видань XIX–XX століть.
# Про проєкт
В архіві зберігаються інформація про рідкіснні та більш сучасні україномовні видання книг для ознайомлення користувачів про різні книги України
## Склад колекції- Кількість записів: 6- Хронологічне охоплення: 1798–2023- Мови документів: українська- Метадані відповідають стандарту: **Dublin Core**
## Переглянути архів онлайн
[Відкрити цифровий архів](https://olinchuk2006.github.io/practical-works/)
## Структура репозиторію
| Файл / Папка | Призначення |
|---|---|
| `index.html` | Головна сторінка каталогу |
| `style.css` | Стилі оформлення |
| `book_N.html` | Детальні сторінки книг |
| `build_site.py` | Python-скрипт генерації сайту |
| `archive_metadata.json` | База даних у форматі JSON (Dublin Core) |
| `books.txt` | Вхідні дані для конвертера |
## Як запустити генерацію локально
1. Переконайтеся, що встановлено Python 3
2. Клонуйте репозиторій:
   ```
   git clone https://github.com/olinchuk2006/practical-works
   ```
3. Запустіть скрипт:
   ```
   python build_site.py
   ```
4. Відкрийте згенерований `index.html` у браузері
## Технології- **Python 3** — генерація HTML із JSON-даних- **HTML / CSS** — розмітка та оформлення- **GitHub Pages** — безкоштовний хостинг статичного сайту- **Dublin Core Metadata Initiative** — стандарт метаданих--
